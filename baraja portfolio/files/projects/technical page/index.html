Welcome To CSS Course

Introduction

CSS is a language of design. We use this to design our web pages efficiently. It has various rules and selectors to apply styling. CSS has evolved within the years and it is used with every web pages to style them.
Here is an example code snippet of css

div#title{
    font-size: 20px;
    opacity: 0.8;
}

Box model

In CSS, we can treat every HTML element as boxes and style them accordingly and this is what CSS does. Whenever we add borders and background colors, we see that every element is surrounded by a rectangular box and all the styling happens within the box.

Dimensions

To measure and decide how long or tall an element should be, we use height and width properties. We have different dimension units in CSS like px, cm, in, em, rem, % etc.

Margin and padding

Margin defines the space an element should have outside the box, while padding defines the space between the border of the element and the content of the box.

Colors

Colors are what gives life to designing. In CSS, we can give colors to text using the color property and background can be set using the background-color property.

Typography

Typography is all about text and its style and decorations. To control typography we have different properties:

  • text-decoration
  • color
  • font-weight
  • font-family
  • font-size

CSS 3

CSS 3 is newly launched feature of CSS with lots of new features. We have border-radius, transform, transition, flexbox, grid and animation. It has become very interesting and intuitive to design and develop in CSS nowadays. CSS 3 has so many other new features that now you can almost do image clipping and filtering of images in the CSS itself. You can also use sprites and background-position them so that we can see different images with a single HTTP request.

Flexboxes

Flex boxes stand for flexible boxes and is a one dimensional layout system in CSS. It can be used very ingenously for making beautiful websites. With flexboxes we can easily center a content and adjust some of the things which were not possible and developers had to use hacks like using floats. It is a responsive design system and it can be scaled well. We have many properties like:

  • display: flex
  • flex-flow
  • flex:
  • align-items
  • justify-content

Grids

Grid is a powerful layout system in CSS. It is a 2 dimensional grid system. Grids can be made responsive using media queries and it is a new feature. It is gaining popularity because it is a part of core technology and not a framework. Grids give us a feature to easily to make columns and rows and is a new way to build layout in contrast to 12-column grid systems used by libraries and frameworks.

    For Parent Element(grid-container)
  • display: grid
  • grid-template-columns
  • grid-template-rows
  • grid-template-areas
  • justify-content
  • align-content
  • align-items
  • justify-items

    • For Child Element(grid items)
  • grid-coloumn
  • grid-row
  • grid-area
  • grid-column-gap
  • grid-row-gap
  • align-self
  • justify-self
